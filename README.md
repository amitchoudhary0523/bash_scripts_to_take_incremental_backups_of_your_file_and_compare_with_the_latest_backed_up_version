**Bash scripts to take incremental backups of your file and compare the current version of a file with the latest backed up version of the same file.**

------------------------------------------------------------------------------------------------------------------------

There are two bash scripts in this repository:

1. **mybackup.sh**: You can use this bash script to take incremental backups of your file. To use this script you may need to change the location of 'backup_dir' as per your needs.

2. **compare_file_with_lastest_backed_up_version.sh**: You can use this bash script to compare the current version of a file with the latest backed up version of the same file (the latest backed up version of the file was made using the bash script - 'mybackup.sh').
